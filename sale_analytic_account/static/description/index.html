Sale Order analytic Account

Overview

This module will create analytic account automatically once quotation is confirmed and then assign it to analytic account field in SO.

Features
Create analytic account automatically once quotation is confirmed. .
Assign the created analytic account to analytic account field in SO.
Use the same analytic account if the SO set back to quotation and reconfirmed again.
How It Works

To use this module, you need first to enable Analytic Accounting from Invoicing -> Configuration -> Settings.
