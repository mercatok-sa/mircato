Odoo Auto Validate POS-Point of sales Session App

Odoo POS Session Auto Validate App helps to Auto Validate and Closed Pos Session through Scheduler/timer/Cron job , user can set specific interval for cron process i.e everyday and based on that all point of sale-pos session validate automatically.

POS Session Scheduler

POS session scheduler will close all open pos session.

Auto Close All Open POS Session

All open pos session will closed and moved to "Closed & Posted".

Track POS Session Log

User can see and track all closed pos session logs.
